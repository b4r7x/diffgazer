# Task T-SEC-HONO — Bump hono + @hono/node-server in published CLI to close 18 advisories

**Source findings:** SEC-001
**Severity:** Critical (CVEs in published `diffgazer@0.1.3`)
**Phase:** 0
**Blocks:** T-PUBLISH-WF (any new publish ships current CVEs)
**Blocked by:** none

## Goal
`pnpm audit --prod --audit-level=moderate` reports 43 advisories. 18 of them flow through the published `diffgazer` CLI via `hono@4.12.0` (16) and `@hono/node-server@1.19.9` (2). Includes:
- `hono` auth bypass via IP spoofing (CVSS 8.2 high)
- `hono` arbitrary file access via serveStatic (CVSS 7.5 high)
- `@hono/node-server` auth bypass (CVSS 7.5 high)

Bump both. The same versions also live in private `cli/server`; mirror to keep server dev-runtime consistent.

## Files to touch (allowlist)
- `cli/diffgazer/package.json` (dependencies)
- `cli/server/package.json` (dependencies)
- `pnpm-lock.yaml` (regenerated by `pnpm install`)

## Files NOT to touch
- `cli/diffgazer/src/**` source — bumps should be ABI-compatible. If any breaking change appears, STOP and report rather than patching source.
- `cli/server/src/**` source — same.

- [ ] `cli/server/package.json` matches
