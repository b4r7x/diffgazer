## This website

Diffgazer docs is a static documentation site. The application does not run analytics scripts, does not set tracking cookies, and does not collect account data through the site itself.

Your browser may still send standard web-server logs to whoever hosts the deployment (for example request IP, user agent, and URL). That logging is controlled by the hosting provider, not by the docs application code.

## The Diffgazer product

Diffgazer runs an embedded local server and reviews your code on your own machine, so your source and API keys never reach a service we run. A network request leaves your computer only when you ask for a review. At that point, the diff and prompt content go to the model provider you chose, under that provider's policy.

The product documentation is the single source of truth for the technical privacy controls and provider request details.

For the full technical walkthrough, see [Privacy and security](/app/concepts/privacy) in the product documentation.
