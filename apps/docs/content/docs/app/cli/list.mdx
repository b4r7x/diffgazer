---
title: dgadd list
description: List available Diffgazer registry items.
---

## Usage

```bash
pnpm exec dgadd list
```

> **Note:** Diffgazer packages are not yet published to npm. Until the first release, install from a local checkout of the repository.

## Examples

```bash
pnpm exec dgadd list
pnpm exec dgadd list --json
pnpm exec dgadd list --installed
```

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--cwd <path>` | Working directory | `.` |
| `--json` | Output as JSON | `false` |
| `--installed` | Show only installed items | `false` |
| `--all` | Include hidden/internal items | `false` |

## Output

Names are shown with their install namespace:

```text
ui/button
ui/dialog
keys/navigation
keys/focus-trap
keys/focus-restore
keys/scroll-lock
```
