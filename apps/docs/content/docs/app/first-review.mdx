# Your first review

From inside a repository with changes you want reviewed, run:

```bash
diffgazer
```

diffgazer prints its banner, starts a local server, and opens the review environment in your browser. Findings render inline using the same diff components the [@diffgazer/ui](/ui) library ships.

When you are done, stop diffgazer from the terminal with `Ctrl+C`.

> diffgazer is local-first: your code and your model keys never leave your machine.
