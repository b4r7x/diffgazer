# diffgazer

diffgazer is a local-first CLI that reviews your code changes with AI. Point it at your working changes and get a structured review where you already work — in a local web environment it serves for you, or in a keyboard-first terminal UI. Your source and your provider keys stay on your machine.

## Highlights

- **Local-first.** Runs as a single global binary. No server to deploy, nothing uploaded to a dashboard you do not control.
- **Bring your own model.** diffgazer is provider-agnostic — you supply the model and the keys.
- **Two ways to review.** A local web environment by default, plus an opt-in keyboard-first terminal UI (beta).
