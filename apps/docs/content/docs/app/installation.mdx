# Installation

diffgazer ships as a single global binary. It requires **Node.js 20 or newer**.

```bash
npm install -g diffgazer
```

Verify the install:

```bash
diffgazer --version
```

Then run it from inside any repository:

```bash
diffgazer
```

By default this starts diffgazer's local web environment and opens it in your browser. See [Your first review](/app/first-review) for what happens next, or [Review modes](/app/review-modes) for the terminal UI.
