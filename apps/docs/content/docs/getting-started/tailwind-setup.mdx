diffui requires Tailwind CSS 4 with the diffui theme variables.

## 1. Install Tailwind 4

```bash
pnpm add -D tailwindcss @tailwindcss/vite
```

## 2. Configure Vite

```ts
// vite.config.ts
import tailwindcss from "@tailwindcss/vite"

export default defineConfig({
  plugins: [tailwindcss()],
})
```

## 3. Import the theme

```css
/* src/index.css */
@import "tailwindcss";
@import "./styles/theme.css";
```

> **Tip**: If you used `npx diffui init`, the theme file is already copied to your project as `styles/theme.css`.

## Theme Variables

diffui uses a two-layer CSS variable system:

**Layer 1 -- Primitives** (`--tui-*`): Raw color values from the GitHub Dark palette.

**Layer 2 -- Semantics**: Functional tokens that reference primitives.

| Primitive | Value (dark) | Semantic tokens |
|-----------|-------------|-----------------|
| `--tui-bg` | `#0d1117` | `--background` |
| `--tui-fg` | `#c9d1d9` | `--foreground` |
| `--tui-blue` | `#58a6ff` | `--primary`, `--accent`, `--ring`, `--info` |
| `--tui-green` | `#3fb950` | `--success` |
| `--tui-red` | `#ff7b72` | `--destructive`, `--error` |
| `--tui-yellow` | `#d29922` | `--warning` |
| `--tui-violet` | `#bc8cff` | (used in syntax highlighting) |
| `--tui-border` | `#30363d` | `--border` |
| `--tui-muted` | `#6e7681` | `--muted` |

See [Theme Overview](/docs/theme/overview) for the full variable diagram.

## Dark Mode

Dark mode is the default. Light mode activates via `[data-theme="light"]`:

```html
<!-- Dark (default) -->
<html>

<!-- Light -->
<html data-theme="light">
```

See [Dark Mode](/docs/theme/dark-mode) for theme switching implementation.

## Font

diffui uses JetBrains Mono for the terminal aesthetic. Add this once in your app `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500;700&display=swap"
/>
```

Apply the base styles to your root element:

```html
<body class="tui-base">
```
