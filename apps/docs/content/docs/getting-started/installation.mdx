## Installation Methods

diffui supports two installation methods: the custom CLI and the shadcn registry.

---

## Option A: diffui CLI

### Quick Start

```bash
npx diffui init
```

This will:
- Detect your project configuration (Tailwind, TypeScript paths)
- Create `components/ui/` directory
- Install dependencies (`clsx`, `tailwind-merge`, `class-variance-authority`)
- Create `src/lib/utils.ts` with the `cn()` utility
- Add `styles/theme.css` to your project

Then add components:

```bash
npx diffui add button
npx diffui add dialog tabs menu
```

### Package Managers

```bash
# pnpm (recommended)
pnpm dlx diffui init

# npm
npx diffui init

# yarn
yarn dlx diffui init

# bun
bunx diffui init
```

## Option B: shadcn Registry

If you already use shadcn/ui, you can install diffui components directly through the shadcn CLI.

### 1. Add registry namespaces

Add the diffui and keyscope registries to your `components.json`:

```jsonc
{
  "registries": {
    "@diffui": { "url": "https://<your-diffgazer-domain>/r/diff-ui/{name}.json" },
    "@keyscope": { "url": "https://<your-diffgazer-domain>/r/keyscope/{name}.json" }
  }
}
```

Replace `<your-diffgazer-domain>` with the domain that serves your docs/registry.

### 2. Install components

```bash
npx shadcn@latest add @diffui/button
npx shadcn@latest add @diffui/dialog tabs menu
```

Components that require keyboard hooks (e.g. `dialog`, `menu`) will automatically install the corresponding keyscope hooks into your `hooks/` directory via `registryDependencies`. No extra step needed.

### 3. Install the theme

```bash
npx shadcn@latest add https://<your-diffgazer-domain>/r/diff-ui/theme.json
```

This copies `styles/theme.css` and `styles/styles.css` to your project.

### Install keyscope hooks standalone

If you want the keyboard hooks without a diffui component:

```bash
npx shadcn@latest add @keyscope/navigation
npx shadcn@latest add @keyscope/focus-trap
npx shadcn@latest add @keyscope/scroll-lock
```

## Manual Installation

### 1. Install dependencies

```bash
pnpm add clsx tailwind-merge class-variance-authority
```

### 2. Create utils file

```ts
// src/lib/utils.ts
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

### 3. Copy component files

Copy the component source from the registry into your project:

```
your-project/
  src/
    components/
      ui/
        button/
          button.tsx
        dialog/
          dialog.tsx
          dialog-trigger.tsx
          dialog-content.tsx
          ...
    lib/
      utils.ts
```

### 4. Import the theme

See [Tailwind Setup](/docs/getting-started/tailwind-setup) for theme configuration.
