---
title: diffui list
description: List all available components in the registry.
---

## Usage

```bash
npx diffui list
```

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--json` | Output as JSON | `false` |
| `--installed` | Show only installed components | `false` |
