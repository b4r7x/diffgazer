---
title: diffui diff
description: Compare local component files with the latest registry versions.
---

## Usage

```bash
npx diffui diff [components...]
```

## Examples

```bash
# Diff all installed components
npx diffui diff

# Diff specific components
npx diffui diff button tabs command-palette
```

## What It Does

1. Loads your `diffui.json` configuration.
2. Resolves requested components (or auto-detects installed ones).
3. Compares each local file against registry content.
4. Prints unified colorized diffs.
5. Shows a summary with changed/unchanged/not-installed counts.

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--cwd <path>` | Working directory | `.` |

## Notes

- If no component names are passed, `diff` checks all installed components.
- If no installed components are found, it exits with a friendly message.
- Header lines (`+++` / `---`) are preserved, while diff hunks are colorized.
