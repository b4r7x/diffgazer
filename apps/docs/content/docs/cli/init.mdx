---
title: diffui init
description: Initialize diffui in your project. Detects your setup and configures everything.
---

## Usage

```bash
npx diffui init
```

## What It Does

1. Detects your project configuration:
   - Package manager (pnpm, npm, yarn, bun)
   - Tailwind CSS version
   - TypeScript path aliases
   - Source directory (`src/` or root)

2. Creates the component directory:
   - `src/components/ui/` (default)
   - Configurable via `--components-dir`

3. Installs dependencies:
   - `clsx` + `tailwind-merge` (class merging)
   - `class-variance-authority` (variant system)

4. Creates utility file:
   - `src/lib/utils.ts` with `cn()` utility (clsx + tailwind-merge)

5. Copies theme file:
   - `styles/theme.css` with all CSS variables

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `--cwd <path>` | Working directory | `.` |
| `--components-dir <path>` | Component install directory | `src/components/ui` |
| `--yes` | Skip confirmation prompts | `false` |
