## Overview

EmptyState is a compound component that displays a centered message when there is no content to show. Use it for empty lists, zero-result searches, and initial states before data loads.

## Web Equivalent

Maps to diff-ui's `EmptyState` component. The terminal version centers content using Ink's flex alignment instead of CSS.

## Import

```tsx
import { EmptyState } from "../../components/ui/empty-state.js";
```

## Props

### EmptyState

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Should contain `EmptyState.Message` and/or `EmptyState.Description` |

### EmptyState.Message

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Bold primary message |

### EmptyState.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Secondary explanatory text in muted color |

## Usage

```tsx
<EmptyState>
  <EmptyState.Message>No reviews yet</EmptyState.Message>
  <EmptyState.Description>
    Start a new review from the home screen.
  </EmptyState.Description>
</EmptyState>
```

## Notes

- The root element centers its children both horizontally and vertically.
- `EmptyState.Message` renders bold text.
- `EmptyState.Description` renders text in the theme's `muted` color.
