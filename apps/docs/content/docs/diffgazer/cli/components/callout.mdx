## Overview

Callout renders a rounded-border box with a leading icon, used for informational messages, warnings, errors, and success confirmations. It is a compound component with `Callout.Title` and `Callout.Content` sub-components.

## Web Equivalent

Maps to diff-ui's `Callout` component. The terminal version uses Ink's `borderStyle="round"` for the box outline instead of CSS borders.

## Import

```tsx
import { Callout } from "../../components/ui/callout.js";
```

## Props

### Callout

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"info" \| "warning" \| "error" \| "success"` | `"info"` | Semantic variant controlling border color and icon |
| `children` | `ReactNode` | -- | Should contain `Callout.Title` and/or `Callout.Content` |

### Callout.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Bold title text |

### Callout.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Body text |

## Usage

```tsx
<Callout variant="warning">
  <Callout.Title>Unsaved changes</Callout.Title>
  <Callout.Content>Your settings have not been saved yet.</Callout.Content>
</Callout>
```

## Variant Icons

Each variant displays a different Unicode icon:

| Variant | Icon |
|---------|------|
| `info` | information symbol |
| `warning` | warning triangle |
| `error` | cross mark |
| `success` | check mark |

## Notes

- The border color and icon color both derive from the variant's token in `useTheme()`.
- `Callout.Title` renders bold text. `Callout.Content` renders normal-weight text.
- The icon and content are laid out in a horizontal row with a gap.
