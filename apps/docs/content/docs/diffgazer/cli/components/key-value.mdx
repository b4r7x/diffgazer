## Overview

KeyValue renders a horizontal label-value pair. The label is displayed in the muted color, followed by a colon and the value in the foreground color. An optional `labelWidth` prop enables column alignment across multiple rows.

## Web Equivalent

No direct diff-ui counterpart. This is a terminal-specific utility component for structured data display, similar to a definition list.

## Import

```tsx
import { KeyValue } from "../../components/ui/key-value.js";
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | -- | The label text |
| `value` | `ReactNode` | -- | The value content |
| `labelWidth` | `number` | -- | Fixed character width for the label (right-padded with spaces) |

## Usage

```tsx
<KeyValue label="Provider" value="Gemini" />
<KeyValue label="Model" value="gemini-2.0-flash" />
```

## Aligned Columns

Use `labelWidth` to align multiple KeyValue rows into a table-like layout:

```tsx
<Box flexDirection="column">
  <KeyValue label="Provider" value="Gemini" labelWidth={12} />
  <KeyValue label="Model" value="gemini-2.0-flash" labelWidth={12} />
  <KeyValue label="Status" value="active" labelWidth={12} />
</Box>
```

This right-pads each label to 12 characters, so the values line up:

```
Provider    :  Gemini
Model       :  gemini-2.0-flash
Status      :  active
```

## Notes

- The label is always rendered in the theme's `muted` token color.
- The value is rendered in the theme's `fg` token color.
- When `labelWidth` is not set, the label occupies its natural width.
