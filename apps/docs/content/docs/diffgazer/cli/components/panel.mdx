## Overview

Panel is a compound component that renders a full-width bordered container. It provides header, content, and footer sections. Use it as the primary layout container for screen content.

## Web Equivalent

Maps to diff-ui's `Panel` or `Card` component. The terminal version uses Ink's `borderStyle="round"` for the container outline.

## Import

```tsx
import { Panel } from "../../components/ui/panel.js";
```

## Props

### Panel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "borderless"` | `"default"` | `default` renders a rounded border; `borderless` omits the border |
| `children` | `ReactNode` | -- | Should contain `Panel.Header`, `Panel.Content`, and/or `Panel.Footer` |

### Panel.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "terminal" \| "subtle"` | `"default"` | `terminal` uses the accent color; `subtle` uses the muted color; `default` inherits |
| `children` | `ReactNode` | -- | Bold header text |

### Panel.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Main content with 1 unit of padding |

### Panel.Footer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Footer text in muted color, preceded by a horizontal rule |

## Usage

```tsx
<Panel>
  <Panel.Header variant="terminal">Review Results</Panel.Header>
  <Panel.Content>
    <Text>3 issues found across 5 files.</Text>
  </Panel.Content>
  <Panel.Footer>Press Enter to view details</Panel.Footer>
</Panel>
```

## Borderless Variant

Use `variant="borderless"` for a flat container without the rounded border:

```tsx
<Panel variant="borderless">
  <Panel.Content>
    <Text>Flat content area</Text>
  </Panel.Content>
</Panel>
```

## Notes

- The default variant renders a full-width box with `borderStyle="round"` and the theme's `border` token color.
- `Panel.Footer` draws a 40-character horizontal rule before the footer text using the `muted` token.
- `Panel.Content` applies 1 unit of padding on all sides.
