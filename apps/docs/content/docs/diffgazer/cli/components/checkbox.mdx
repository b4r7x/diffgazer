title: "CheckboxGroup"
description: "Multi-select checkbox list with keyboard navigation and toggle."

## Overview

CheckboxGroup is a compound component providing a multi-select list. Items display `[x]` or `[ ]` indicators. Users navigate with arrow keys and toggle selection with Space. Supports controlled and uncontrolled modes.

## Web Equivalent

Maps to diff-ui's `Checkbox` component. The terminal version uses ASCII indicators (`[x]`/`[ ]`) and arrow-key navigation instead of clickable checkboxes.

## Import

```tsx
import { CheckboxGroup } from "../../components/ui/checkbox.js";
```

## Props

### CheckboxGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `T[]` | -- | Controlled selected values |
| `defaultValue` | `T[]` | -- | Initial selected values (uncontrolled mode) |
| `onChange` | `(value: T[]) => void` | -- | Called when selection changes |
| `onHighlightChange` | `(value: string) => void` | -- | Called when the highlighted item changes |
| `wrap` | `boolean` | `true` | Wrap navigation from last to first item and vice versa |
| `disabled` | `boolean` | `false` | Disable all items |
| `isActive` | `boolean` | `true` | Whether the group accepts keyboard input |
| `children` | `ReactNode` | -- | Should contain `CheckboxGroup.Item` elements |

The generic type parameter `T` defaults to `string` and constrains `value`/`defaultValue` arrays.

### CheckboxGroup.Item

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | -- | Unique value identifying this item |
| `label` | `ReactNode` | -- | Display label |
| `description` | `ReactNode` | -- | Optional secondary text below the label |
| `disabled` | `boolean` | `false` | Disable this individual item |

## Usage

```tsx
const [selected, setSelected] = useState<string[]>([]);

<CheckboxGroup value={selected} onChange={setSelected} isActive>
  <CheckboxGroup.Item value="lint" label="Run linter" />
  <CheckboxGroup.Item value="typecheck" label="Type check" />
  <CheckboxGroup.Item value="test" label="Run tests" description="Executes vitest" />
</CheckboxGroup>
```

## Keyboard Interaction

| Key | Action |
|-----|--------|
| Up Arrow | Move highlight to previous item |
| Down Arrow | Move highlight to next item |
| Space | Toggle the highlighted item's checked state |

## Visual Indicators

- `[x]` -- checked item
- `[ ]` -- unchecked item
- Highlighted item is bold and colored with the accent token
- Disabled items are dimmed
- Descriptions are rendered in the muted color, indented below the label

## Notes

- Uses React Context internally. `CheckboxGroup.Item` must be a direct child of `CheckboxGroup`.
- When `wrap` is `true` (default), navigating past the last item wraps to the first.
- Disabled items are skipped during keyboard navigation.
