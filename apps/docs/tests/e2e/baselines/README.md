# Visual baselines

Per-platform PNG snapshots generated by Playwright `expect(...).toHaveScreenshot()`.
This directory is intentionally empty until baselines are generated.

## Generate baselines (operator, local Linux/CI image)

```bash
pnpm --filter @diffgazer/docs exec playwright install --with-deps chromium
pnpm --filter @diffgazer/docs build
pnpm --filter @diffgazer/docs test:e2e -- --update-snapshots
```

Baselines are platform-sensitive. Generate on the same OS/browser image used in CI
(Linux + chromium in `ubuntu-latest`) and commit the resulting `*.png` files in this directory.

## Run against existing baselines

```bash
pnpm --filter @diffgazer/docs test:e2e
```

Tolerance for pixel diffs is configured via `expect.toHaveScreenshot.maxDiffPixelRatio: 0.01`
in `apps/docs/playwright.config.ts`.
