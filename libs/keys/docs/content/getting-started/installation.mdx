`@diffgazer/keys` is a hooks-first keyboard accessibility toolkit for React.

## Package Install

Use the package when you need provider-backed hooks such as `KeyboardProvider`, `useKey`, `useScope`, `useScopedNavigation`, `useActionRowNavigation`, or `useFocusZone`.

`@diffgazer/keys` is external publish-gated as of May 6, 2026. Use the npm command only after `npm view @diffgazer/keys version` succeeds, or validate with a locally packed workspace tarball.

```bash
npm install @diffgazer/keys
```

```tsx
import { KeyboardProvider, useKey } from "@diffgazer/keys"
```

## Copy Standalone Hooks

Use `dgadd` when you want source-owned standalone hooks with no package runtime dependency.

```bash
pnpm exec dgadd init
pnpm exec dgadd add keys/navigation
pnpm exec dgadd add keys/focus-trap keys/focus-restore keys/scroll-lock
```

`@diffgazer/add` is also publish-gated. Before publication, install a local `@diffgazer/add` tarball into a fixture app and run `pnpm exec dgadd`. Use `npx @diffgazer/add ...` only after `npm view @diffgazer/add version` succeeds.

The copied hooks are installed into your configured hooks directory, usually `src/hooks`.

## Direct shadcn / Manual Copy (future, after publication)

The hosted registry at `https://r.b4r7.dev/` is not yet live. After publication, public keys registry items will be installable directly with shadcn:

```bash
npx shadcn add https://r.b4r7.dev/keys/navigation.json
npx shadcn add https://r.b4r7.dev/keys/focus-trap.json
```

Until then, use `pnpm exec dgadd add keys/navigation` or copy source directly from `https://github.com/b4r7x/diffgazer/tree/main/libs/keys/src/hooks`.

Files install to `src/hooks/use-navigation.ts`, `src/hooks/utils/navigation-items.ts`, etc.

## API Availability by Path

Standalone hooks available through all three paths (package, dgadd, direct copy):

- `useNavigation`, `useFocusRestore`, `useFocusTrap`, `useScrollLock`

Package-only APIs (require `KeyboardProvider`, not available through copy paths):

- `KeyboardProvider`, `useKey`, `useScope`, `useScopedNavigation`, `useActionRowNavigation`, `useFocusZone`, `keys`, `useKeyboardContext`, `useOptionalKeyboardContext`

## With @diffgazer/ui

Keyboard-enabled UI components can install required hooks automatically:

```bash
pnpm exec dgadd add ui/menu --integration copy
pnpm exec dgadd add ui/dialog --integration keys
```

`copy` mode writes local hook source. `keys` mode imports from the `@diffgazer/keys` package.
