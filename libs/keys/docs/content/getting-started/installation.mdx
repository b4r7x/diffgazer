`@diffgazer/keys` is a hooks-first keyboard accessibility toolkit for React.

## Package Install

Use the package when you need provider-backed hooks such as `KeyboardProvider`, `useKey`, `useScope`, or `useFocusZone`.

`@diffgazer/keys` is external publish-gated as of May 6, 2026. Use the npm command only after `npm view @diffgazer/keys version` succeeds, or validate with a locally packed workspace tarball.

```bash
npm install @diffgazer/keys
```

```tsx
import { KeyboardProvider, useKey } from "@diffgazer/keys"
```

## Copy Standalone Hooks

Use `dgadd` when you want source-owned standalone hooks with no package runtime dependency.

```bash
npx @diffgazer/add init
npx @diffgazer/add add keys/navigation
npx @diffgazer/add add keys/focus-trap keys/scroll-lock
```

`@diffgazer/add` is also publish-gated. Before publication, install a local `@diffgazer/add` tarball into a fixture app and run `pnpm exec dgadd`.

The copied hooks are installed into your configured hooks directory, usually `src/hooks`.

## With @diffgazer/ui

Keyboard-enabled UI components can install required hooks automatically:

```bash
npx @diffgazer/add add ui/menu --integration copy
npx @diffgazer/add add ui/dialog --integration keys
```

`copy` mode writes local hook source. `keys` mode imports from the `@diffgazer/keys` package.
