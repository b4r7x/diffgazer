`@diffgazer/keys` is an installable React package. Hook source installation is handled by `dgadd`, the same installer used for `@diffgazer/ui`.

## Usage

```bash
pnpm exec dgadd add keys/<hook>
```

`@diffgazer/add` and `@diffgazer/keys` are external publish-gated as of May 6, 2026. Before publication, install locally packed tarballs into the target app and run `pnpm exec dgadd`. Use `npx` and `npm install` commands only after `npm view` returns versions.

## Commands

| Command | Purpose |
|---------|---------|
| `pnpm exec dgadd init` | Initialize `diffgazer.json` |
| `pnpm exec dgadd add keys/<hook>` | Copy a standalone hook into your project |
| `pnpm exec dgadd list` | List available `keys/*` hooks |
| `pnpm exec dgadd diff keys/<hook>` | Compare installed hook files |
| `pnpm exec dgadd remove keys/<hook>` | Remove installed hook files |

For package-mode imports, install `@diffgazer/keys` directly:

```bash
npm install @diffgazer/keys
```
