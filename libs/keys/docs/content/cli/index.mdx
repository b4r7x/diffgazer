`@diffgazer/keys` is an installable React package. Hook source installation is handled by `dgadd`, the same installer used for `@diffgazer/ui`.

## Usage

```bash
npx @diffgazer/add add keys/<hook>
```

`@diffgazer/add` and `@diffgazer/keys` are external publish-gated as of May 6, 2026. Use `npx` and `npm install` commands only after `npm view` returns versions, or validate with locally packed tarballs.

## Commands

| Command | Purpose |
|---------|---------|
| `npx @diffgazer/add init` | Initialize `diffgazer.json` |
| `npx @diffgazer/add add keys/<hook>` | Copy a standalone hook into your project |
| `npx @diffgazer/add list` | List available `keys/*` hooks |
| `npx @diffgazer/add diff keys/<hook>` | Compare installed hook files |
| `npx @diffgazer/add remove keys/<hook>` | Remove installed hook files |

For package-mode imports, install `@diffgazer/keys` directly:

```bash
npm install @diffgazer/keys
```
