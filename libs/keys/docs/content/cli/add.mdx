## Copy Standalone Hooks

```bash
npx @diffgazer/add add keys/navigation
npx @diffgazer/add add keys/focus-trap keys/scroll-lock
```

Standalone hooks are copied into your project as source files and have no runtime dependency on `@diffgazer/keys`.

## Use The Package

Provider-based hooks such as `useKey`, `useScope`, and `useFocusZone` come from the package:

```bash
npm install @diffgazer/keys
```

This package install is publish-gated until `npm view @diffgazer/keys version` succeeds.

```tsx
import { KeyboardProvider, useKey } from "@diffgazer/keys"
```

UI components can also be installed in package integration mode:

```bash
npx @diffgazer/add add ui/menu --integration keys
```
