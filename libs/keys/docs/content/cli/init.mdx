---
title: dgadd init
description: Initialize installer configuration before adding @diffgazer/keys hooks.
---

```bash
pnpm exec dgadd init
```

Before publication, install a locally packed `@diffgazer/add` tarball into the target app and run `pnpm exec dgadd`. Use `npx @diffgazer/add init` only after `npm view @diffgazer/add version` succeeds.

This creates `diffgazer.json` with aliases for components, hooks, and utilities. Hooks are installed into the configured hooks directory, usually `src/hooks`.
