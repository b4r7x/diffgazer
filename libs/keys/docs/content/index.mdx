# @diffgazer/keys

Reusable keyboard-first behavior for React: scopes, key registration, list navigation, focus zones, focus trap and restore, scroll lock, and small keyboard and focus helpers. No runtime dependencies. React `>=19.2.0` is a peer dependency.

Standalone hooks (`useNavigation`, `useFocusTrap`, `useFocusRestore`, `useScrollLock`) work without a provider and can be copied through `dgadd` or the public registry. Provider-backed APIs (`KeyboardProvider`, `useKey`, `useScope`, `useScopedNavigation`, `useActionRowNavigation`, `useFocusZone`) are package-only and require `KeyboardProvider`.
