Quick map of the main public exports from `@diffgazer/keys`. Requires React `>=19.2.0`.

> See the [interactive playground](https://github.com/b4r7x/diffgazer/tree/main/libs/keys/examples/playground) for live demos of each hook.

---

## Provider

| Export | Description |
|--------|-------------|
| [KeyboardProvider](/keys/api/keyboard-provider) | Context provider that enables scoped keyboard handling. Wraps your app or subtree. |

## Hooks

| Hook | Category | Description |
|------|----------|-------------|
| [useKey](/keys/hooks/use-key) | Provider | Bind keyboard shortcuts with scoped, document-level, or container-scoped listening. |
| [useScope](/keys/hooks/use-scope) | Provider | Push a named scope onto the keyboard scope stack. |
| [useNavigation](/keys/hooks/use-navigation) | Standalone | Standalone keyboard navigation for role-based lists and tabs. |
| [useScopedNavigation](/keys/hooks/use-scoped-navigation) | Provider | Scope-aware keyboard navigation registered via KeyboardProvider. |
| [useActionRowNavigation](/keys/hooks/use-action-row-navigation) | Provider | Two-zone keyboard navigation for rows with inline actions. |
| [useFocusZone](/keys/hooks/use-focus-zone) | Provider | Manage focus across multiple zones with arrow key and Tab transitions. |
| [useFocusTrap](/keys/hooks/use-focus-trap) | Standalone | Trap Tab focus within a container element. |
| [useFocusRestore](/keys/hooks/use-focus-restore) | Standalone | Capture and restore focus around temporary UI with nested stack safety. |
| [useScrollLock](/keys/hooks/use-scroll-lock) | Standalone | Prevent body/element scrolling with reference counting. |
