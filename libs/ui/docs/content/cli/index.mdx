The `dgadd` CLI installs source-owned `@diffgazer/ui` components and `@diffgazer/keys` hooks into your app.

`@diffgazer/add` is external publish-gated as of May 6, 2026. Use local packed tarballs for validation until `npm view @diffgazer/add version` succeeds.

## Usage

```bash
dgadd <command> [options]
```

You can also run it without a global install:

```bash
npx @diffgazer/add <command> [options]
```

Use `npx @diffgazer/add` only after publication. Before that, install a local tarball and run `pnpm exec dgadd`.

## Commands

| Command | Purpose | Docs |
|---------|---------|------|
| `init` | Initialize `diffgazer.json` in a project | [/ui/docs/cli/init](/ui/docs/cli/init) |
| `add` | Add `ui/*` components or `keys/*` hooks | [/ui/docs/cli/add](/ui/docs/cli/add) |
| `list` | List available or installed items | [/ui/docs/cli/list](/ui/docs/cli/list) |
| `diff` | Compare local files with bundled registry versions | [/ui/docs/cli/diff](/ui/docs/cli/diff) |
| `remove` | Remove installed items safely | [/ui/docs/cli/remove](/ui/docs/cli/remove) |

## Global Option

| Flag | Description |
|------|-------------|
| `-s, --silent` | Suppress non-error output |
