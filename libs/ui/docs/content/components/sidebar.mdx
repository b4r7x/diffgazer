## Variants

Six active-marker variants — caret, inverted, bar, bracket, block, and terminal — share row metrics so glyph slots stay aligned. Switch between them below to compare the active state and the prominent section-title style. `terminal` is the TUI rail: a continuous 1px hairline down the rows, a `>` prompt on the active item, and no background fill.
