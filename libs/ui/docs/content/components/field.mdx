## Composition

Use `Field` for form wiring: label, control id, required, disabled, invalid, description, error, and ARIA relationships.

`Input` stays the bare native input wrapper. `InputGroup` is only a decorated input shell for prefix and suffix content. Compose `Field` with `Input`, `InputGroup`, `Textarea`, `Select`, or another control instead of turning decorated inputs into standalone form fields.
