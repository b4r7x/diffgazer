## Figlet Integration

The optional `getFigletText` helper generates ASCII art at runtime using the
[figlet](https://www.npmjs.com/package/figlet) library. It is available at
`@diffgazer/ui/components/logo/figlet` (npm package mode only) and supports
two fonts: `"Big"` and `"Small"`.

```tsx
import { getFigletText } from "@diffgazer/ui/components/logo/figlet"

const ascii = await getFigletText("DG", "Big")
```

`figlet` is an optional peer dependency. Install it with
`npm install figlet @types/figlet` before using `getFigletText`.
