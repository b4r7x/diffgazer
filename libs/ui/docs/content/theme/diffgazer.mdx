## Why Diffgazer

Diffgazer gives solo developers a local-first code review loop:

- review requests send diff and prompt content to the provider you choose
- independent review lenses focus on different failure modes
- review history and issue drilldowns keep context in one place

## Quick Start

The public `diffgazer` package is published. Run `npm view diffgazer version engines --json` to verify the current published version and Node floor, then install it globally:

```bash
npm install -g diffgazer
cd your-project
diffgazer
```

More details: [github.com/b4r7x/diffgazer](https://github.com/b4r7x/diffgazer)
