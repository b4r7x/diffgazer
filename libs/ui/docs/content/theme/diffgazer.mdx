## Why Diffgazer

Diffgazer gives solo developers a local-first code review loop:

- only your `git diff` is sent to the provider you choose
- independent review lenses focus on different failure modes
- review history and issue drilldowns keep context in one place

## Quick Start

The public `diffgazer` package is future deployment work. Use this global install command only after its npm package is published:

```bash
npm install -g diffgazer
cd your-project
diffgazer
```

More details: [github.com/b4r7x/diffgazer](https://github.com/b4r7x/diffgazer)
