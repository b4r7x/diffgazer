## 0.1.0 (2026-02-XX)

Initial release.

### Components (28)

Badge, Block Bar, Button, Callout, Card, Checkbox, Checklist, Code Block,
Dialog, Diff View, Empty State, Focusable Pane, Horizontal Stepper, Input,
Key Value, Key Value Row, Labeled Field, Menu, Navigation List, Panel,
Radio Group, Scroll Area, Search Input, Section Header, Stepper, Tabs,
Toast, Toggle Group.

### Features

- CLI: `dgadd init`, `add`, `list`
- Two-layer CSS variable theme system (GitHub Dark palette)
- Dark mode default with `[data-theme]` switching
- All components use Tailwind 4 + CVA
- No keyboard handling baked in -- your app, your keyboard logic
