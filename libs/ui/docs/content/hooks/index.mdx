---
title: Hooks
description: Headless hooks that power @diffgazer/ui components and can be reused directly.
---

Reusable hooks for state, positioning, overflow, and focus behavior. Each is installable through `dgadd`.

- [active-heading](/ui/hooks/active-heading) — track the heading in view for a table of contents
- [controllable-state](/ui/hooks/controllable-state) — controlled and uncontrolled value state
- [floating-indicator](/ui/hooks/floating-indicator) — animate an indicator across active items
- [floating-position](/ui/hooks/floating-position) — position floating content against an anchor
- [form-reset](/ui/hooks/form-reset) — run a callback when an enclosing form resets
- [listbox](/ui/hooks/listbox) — active-descendant listbox navigation and typeahead
- [outside-click](/ui/hooks/outside-click) — detect clicks outside an element
- [overflow-detection](/ui/hooks/overflow-detection) — detect when content overflows its container
- [overflow-items](/ui/hooks/overflow-items) — partition items into visible and overflow sets
- [presence](/ui/hooks/presence) — keep elements mounted through exit animations
