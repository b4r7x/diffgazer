# @diffgazer/ui

Terminal-inspired UI components for React. Like shadcn/ui but with a TUI aesthetic.

## Supported browsers

The library inherits the Tailwind CSS v4 baseline:

| Browser | Minimum version |
|---------|-----------------|
| Chrome  | 111             |
| Edge    | 111             |
| Safari  | 16.4            |
| Firefox | 128             |

A few visual details degrade within that floor: Safari 16.4 - 18.1 ignores `scrollbar-gutter` (gated behind `@supports`); Safari 16.4 - 17.x renders a solid dialog backdrop instead of a blurred one (`-webkit-backdrop-filter` is emitted alongside the standard property). See the [README](https://github.com/b4r7x/diffgazer/tree/main/libs/ui#supported-browsers) for the full matrix.
